# About Me

Repositório do meu site pessoal, desenvolvido para apresentar meu perfil profissional, stack e projetos.

## Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Framer Motion**
- **Tailwind CSS**

## Como executar o projeto

### Pré-requisitos

- Node.js **21+**
- npm

### Clonando o repositório

```bash
git clone https://github.com/jeronimofjr/about-me.git
```

**Acesse o diretório do projeto:**

```bash
cd /about-me
```

### Instalando as dependências

```bash
npm install
```

### Rodando em ambiente de desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em http://localhost:5173.
